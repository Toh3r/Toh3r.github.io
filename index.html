Facial Recognition Using a UAV


Description

I gave a UAV the ability to recognise a specific persons face, fly towards that person and then fly away.

Facial recognition was achieved using openCV and the LBPH (Local Binary Pattern Histogram) algorithm. To train the classifier, facial detection captured 100 images of a persons face. These images are then turned into histograms. When searching for a face the UAV will break the image into a histogram anytime a face is detected. The histograms are then compared and a confidence value is created. If this value is high enough the UAV moves towards that person.

This application also contained a page which allowed the UAV's movements to be controlled from the application.

LSH Shepherding Algorithms


Description

This simulation was created as part of my Masters degree. My masters focused on the area of shepherding algorithms, which is the manipulation of emergent behaviour of group of agents by another agent

I created a number of algorithms that can be used to herd animals with a UAV/UAV's and tested them using this simulation. Animal behaviours are modelled using Reynolds flocking algorithm and UAV movements are dictated by creating target points based on the positions and behaviours of the animal agents. The UAV's movements are based on LSH principals, which are a set of principals created by animal scientists to move farm animals with minimal agitation to the animals.

This project also introduced some new methods of agent location identification and object avoidane that previously had not been implemented in any shepherding algortihm simulations.
