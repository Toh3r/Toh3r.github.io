Can be used instead of non-semantic tags when possible(divs)

More descriptive than divs and create cleaner code

  • <header> - Header Section
  • <nav> - For navigation links
  • <section> - Group related code
  • <main> - Main body of content
  • <article> - Self Contained Content
  • <aside> - For content related to main content or article
  • <figure> - Images
  • <figcaption> - caption for figure tag
  • <details> - Yup
  • <summary> - Heading for details tag
  • <time>
  • <mark> - For highlighted areas
  • <footer> - Bottom of page
