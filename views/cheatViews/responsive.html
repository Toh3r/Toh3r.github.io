CSS Units

Absolute Units - Pixels - Stay static in size

Percentage Units - Relative to size of container, 100% width = 100% of containers width
Use for containers, images, block elements

Relative Units - EM, REM (Relative to font size), VH, VW (Realative to viewport size)
1em/rem = 100%, 1.5 = 150& etc...
em - Relative to parents font-size (defualt is 16px), use for padding/margin
rem - Relative to the root of the document, use for font size
Set root font size (in html tag) to 62.5% (10px) for easy maths

Ems and Rems

            .html {
                font-size: 62.5%; /* Set root font-size to 10px */
            }

            .parent {
                margin: 1.25em;
                padding: 0.75em;
            }

            .child {
                font-size: 1.25rem;
            }
Responsive Containers

            .container {
                width: 90%; /* Will take up 90% of viewport/container */
                max-width: 650px; /* But will not excede 650px */
            }
Responsive Images

            img {
                width: 100%; /* 100% width of container */
                /* Set height to auto, or just don't set it */
            }
Media Querys

Target different types of media, can change css depending on device/device properties

Screen size

            body {
                background: red;
            }

            /* When screen width > than 400px change background to blue */
            /* Don't need to declare screen */
            @media screen and (max-width: 600px) {
                body { /* Declare selector */
                    display-flex:; 
                }
            }

            /* When screen too small, let columns stack */
            .col {
                display: flex;
            }

            @media (max-width: 600px) {
                .col {
                    /* Switches main axis */
                    flex-direction: column;
                }
            }

FlexBox
Basics

Stop divs stacking, fit content thats inside them on horizontal axis
Set parent to display: flex

Flexbox Properties

            .parent {
                display: flex; /* Affects all child elements */
            }
            .child {
                display: flex; 
                flex-wrap: wrap; /* Wrap elements */
                justify-content: center; /* Center elements horizontally */
                justify-content: space-between; /* Spacing */
                justify-content: space-around; /* Spacing */
                justify-content: space-evenly; /* Spacing */
                align-items: center; /* Center elements vertically */
                align-items: flex-start; /* Align at top of parent */
            }
CSS Grid
Set parent to display:grid;
Tracks - columns/rows
Cell - One section of a grid
Grid Area - A combination of cells, must be rectangular

Flexbox Properties

            .parent {
                display: grid; /* Affects all grid elements (child divs) */
            }
            .child {
                grid-template-columns: 150px 100px 125ox; /* Set number of columns and width of columns */
                grid-template-columns: repeat (3, 200px); /* 200px for all columns */
                grid-template-columns: 1fr 3fr 2fr;  /* Fraction of grid to take up */
                grid-column gap: 20px; /* Margin between columns */
                /* Resize columns depending on screen size */
                grid-template-columns: repeat(auto-fit, minmax(200px,1fr));
                grid-column-start: 1; /* Set Starting Column */
                grid-column-end: 3; /* Set Ending Column */
                grid-column: 1 / 3; /* Start/End on same line */
                grid-column-end: span 2; /* Take up 2 cols */
            }
